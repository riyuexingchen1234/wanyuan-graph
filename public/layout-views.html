布局算法

  • 坐标系：X=左右（右+），Y=上下（上+），Z=前后（前+，靠近相机为+Z）。相机位于+Z处看向-Z方向。
  • 产业节点：位于 (0, Hind, 0)，在主链几何中心正上方，最靠前（z=0）。
  • 主链：沿X轴均匀排列在 z=-Dmain 平面，y=0，中心节点对齐 x=0。
支链黄金角螺旋（以父节点p为原点，-Z为后）：
goldenAngle = π·(3−√5) ≈ 2.3999 rad（137.5°）
第k个子节点（k=0,1,2...）：
  zk = p.z − Dlayer·(k+1)    // 每子后退一步
  rk = R0·√(k+1)·shrinkhop    // 半径缓慢增长，越深层收缩
  θk = θstart + k·goldenAngle    // 每步黄金角旋转
  xk = p.x + rk·cos(θk)
  yk = p.y + rk·sin(θk)

上下分配：上游辅材 θstart=π/2（从正上方开始螺旋），下游应用 θstart=3π/2（从正下方开始螺旋）。
深度收缩：每多一层 hop，起始半径R0乘以shrink，保证纵深节点向中心收束。
  • 节点大小：所有节点世界坐标下半径统一为0.05（与未命名.html一致的小球），仅依靠透视近大远小和灰度体现层次。
  • 灰度分级（朴素黑白）：产业#111 → 主链#333 → hop1#777 → hop2#aaa → hop3#ccc → hop≥4#e0e0e0，越远越淡。
  • 连线：主链直线黑色较实；产业→主链直线灰色；父→支链灰色浅淡；越远越淡。
